# fetrust
a minimal and fast system info program
<br>
## looks like:
![image psd](https://github.com/19atlas/fetrust/assets/67108558/64340c6b-c2df-4839-bbed-bbf109a55ae7)

## [Just]
### Install
```sh
just i
```

### Uninstall
```sh
just u
```

### All Available list
```sh
just -l
```

## [Make]
### For build and install: 
```
make build && sudo make install
```
## For delete:
```
sudo make uninstall
```
