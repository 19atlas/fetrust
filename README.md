# fetrust
a minimal and fast system info program

## [Just]

### All Available list
```sh
just -l
```

### Install
```sh
just i
```

### Uninstall
```sh
just u
```

## [Make]

### For build and install: 
```
make build && sudo make install
```
## For delete:
```
sudo make uninstall
```
